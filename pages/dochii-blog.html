🎉 欢迎使用 DOCHII Blog Manager

这是一个由 DOCHII Team 开发的博客管理系统。

✨ 主要功能

  • 📝 Markdown 文档编辑
  • 🌐 自动生成静态网站
  • 🎨 主题系统支持
  • 📚 文档管理

🚀 开始使用

您可以通过以下步骤开始创建您的博客：

  1. 在左侧文件树中选择 markdown 文件夹
  2. 创建新的 .md 文件
  3. 编写您的博客内容
  4. 保存文件后系统会自动生成对应的 HTML 页面

💡 提示：您可以在配置文件中修改网站信息、主题设置等。
